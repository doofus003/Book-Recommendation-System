# Full-Stack Recommendation System

## Overview
A full-stack recommendation system built with FastAPI, React, Docker, and Python.
The system processes a Kaggle CSV dataset and provides recommendations via a REST API and web interface.

## Tech Stack
- Backend: FastAPI
- Frontend: React
- Data: CSV (Kaggle)
- ML: Scikit-learn
- Deployment: Docker & Docker Compose

## Dataset
- Source: Kaggle
- Raw data: `data/data.csv`
- Cleaned data: `data/data_cleaned.csv`
